# Crystal_Morphology
Python workflow for crystal morphology, PCA, K-means clustering and crystal size analysis of host–enclave pairs from the Karadağ Volcanic Complex (Central Anatolia, Türkiye).

Crystal Morphology Analysis

Python workflow accompanying the manuscript:

Gencoglu Korkmaz (2026)

This repository contains the Python workflow used for quantitative crystal morphology analysis of host–enclave pairs from the Karadağ Volcanic Complex (Central Anatolia, Türkiye).

Methods

The workflow performs:

- Principal Component Analysis (PCA)
- K-means clustering
- Silhouette analysis
- Crystal morphology statistics
- Feret diameter analysis
- Summary table generation
- Publication-quality figure generation

Input

CSV files exported from ImageJ.

Output

The workflow automatically generates:

- PCA plots
- K-means clustering plots
- Feret diameter boxplots
- Crystal morphology summary tables
- Silhouette statistics

Required Python packages

- pandas
- numpy
- matplotlib
- scikit-learn
